# Three-Year Roadmap

A structured plan to transition from academic behavioral economist to 
Senior Applied Economist with AI/ML focus, suitable for roles at top 
technology firms and economic consulting.

## Phase Overview

- Year 1: Foundation and Academic-Behavioral Signal
- Year 2: Industry Pivot and Causal Inference Mastery
- Year 3: Senior-Tier Polish and Strategic Positioning

## Year 1 (Months 1-12)

### Months 1-2: Python Foundations (Days 1-20)
Variables, logic, lists, dictionaries, loops, pandas DataFrames, 
filtering, feature engineering, data inspection, basic cleaning, 
visualization, GitHub workflow with branches and pull requests, 
SQL introduction (Day 15-20).

### Months 3-4: Statistical Analysis (Days 21-40)
Descriptive statistics, hypothesis testing, linear regression with 
statsmodels, logistic regression, coefficient interpretation, 
behavioral variable construction.

### Months 5-6: Machine Learning Foundations (Days 41-60)
Train-test splits, cross-validation, classification, model evaluation, 
feature importance, regularization, ML model for financial fragility.

### Months 7-8: Capstone Project 1 Completion (Days 61-80)
Financial Fragility analysis on real SCF data, behavioral proxies, 
regression and ML comparison, SHAP interpretation, policy framing, 
research paper draft, portfolio version.

### Month 9: BSQE Course Repository
Second GitHub repository with lecture-ready modules, problem sets, 
and a model research project for student reference.

## Year 2 (Months 13-24)

### Months 13-15: Project 2 — Causal Marketing Attribution
Difference-in-differences, synthetic control, instrumental variables, 
geographic experiments, marketing attribution, ROI calculations.

### Months 16-18: Project 3 — Pricing and Demand Estimation
Demand estimation, price elasticity, consumer heterogeneity, 
structural modeling, counterfactual simulation.

### Months 19-21: First Paper Submission and Causal ML Module
Double machine learning, causal forests, meta-learners, heterogeneous 
treatment effects, first working paper submitted to a journal.

### Months 22-24: Production Engineering Module
Object-oriented Python, code testing, reproducible environments, 
cloud platform basics, Docker fundamentals, refactoring Project 1.

## Year 3 (Months 25-36)

### Months 25-27: Project 4 — A/B Testing Framework
Experimentation methodology, CUPED variance reduction, sequential 
testing, switchback experiments, multi-armed bandits.

### Months 28-30: Project 5 — Behavioral Default Prediction
Credit risk modeling, survival analysis, behavioral feature 
engineering, causal interpretation, regulatory framing.

### Months 31-33: AI/LLM Integration Module
LLM applications in economic analysis, embedding-based methods, 
sentiment and topic modeling, reinforcement learning conceptual 
foundations.

### Months 34-36: Communication, Networking, and Job Market 
Preparation
Executive memo writing, business case framing, mock interviews, 
LinkedIn rebuild, technical blog or Substack with curated posts, 
conference participation, strategic networking.

## Public Visibility Strategy

### Phase 1 (Months 1-6): Quiet Foundation
LinkedIn profile update, strategic connections with applied economists 
at target firms, thoughtful engagement with 1-2 posts per week.

### Phase 2 (Months 7-12): First Public Voice
One monthly LinkedIn post on capstone concepts, optional Substack 
launch at end of Year 1.

### Phase 3 (Months 13-30): Sustained Professional Presence
Two to three LinkedIn posts per month, one to two Substack posts per 
month, direct engagement with target firm employees.

### Phase 4 (Months 31-36): Strategic Job Market Visibility
Weekly LinkedIn posts, two Substack posts per month, all polished 
and aligned to target roles.

## Skills Inventory

By Year 3, this roadmap delivers:

- Python (pandas, NumPy, statsmodels, scikit-learn)
- SQL (queries, joins, aggregations, window functions)
- Causal inference toolkit (DiD, synthetic control, IV, double ML, 
  causal forests)
- Machine learning (supervised, classification, regression, model 
  interpretation)
- Experimentation methodology
- Production-grade code (OOP, testing, reproducible environments)
- LLM and AI applications
- Executive communication
- Five complete public projects
- Working papers
- BSQE course repository

## Review Cadence

Quarterly strategic review. Each review assesses:
- Progress against plan
- Skill gaps that emerged
- Market changes in target roles
- Realistic adjustments to timeline

The plan is robust to partial completion. Sixty to seventy percent 
completion still produces a dramatically improved profile compared 
to the baseline.
